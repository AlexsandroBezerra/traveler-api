## 💁 About the project

This api provides everything needed to Traveler Web App.

You can list and search for a Ceara cities.

To see the **Traveler web client**, [click here](https://github.com/AlexsandroBezerra/traveler-web) <br />

## 💻 Getting started

### Requirements

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

**Clone the project and access the folder**

```bash
$ git clone https://github.com/AlexsandroBezerra/traveler-api.git && cd traveler-api
```

**Follow the steps below**

```bash
$ docker-compose up -d

# Well done, project is started!
```

## 🤔 How to contribute?
**Make a fork of this repository**

```bash
# Fork using GitHub official command line
# If you don't have the GitHub CLI, use the web site to do that.

$ gh repo fork AlexsandroBezerra/traveler-api
```

**Follow the steps below**

```bash
# Clone your fork
$ git clone your-fork-url && cd traveler-api

# Create a branch with your feature
$ git checkout -b my-feature

# Make the commit with your changes
$ git commit -m 'feat: My new feature'

# Send the code to your remote branch
$ git push origin my-feature
```

After your pull request is merged, you can delete your branch
